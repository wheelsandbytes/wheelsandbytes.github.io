### The easy way to get a Playstation 1 emulator setup on a Mac ![Alt text](img/openemu.png) First, grab [OpenEmu](http://openemu.org) To install the mednafen core, go to OpenEmu > Preferences > Library > Available Libraries > Sony Playstation: ![Alt text](img/check-sony.png) ![Alt text](img/install-mednafen.png) #### Go find some PS1 BIOS files. I'm not linking them for obvious reasons, but here are some links that might be helpful... [Check out good ole YouTube (and read some of the comments)](https://www.youtube.com/watch?v=4ncTG2Vq8bQ") it keeps saying I need "scph 5502.bin" but whenever i drag it it just keeps telling me i need it:/ This one worked for me <a href="http://psxdev.net/downloads/bios/psx/SCPH5502.BIN">http://psxdev.net/downloads/bios/psx/SCPH5502.BIN</a> I downloaded my bios from here: <a href="http://www.emuparadise.me/biosfiles/PSX.zip">http://www.emuparadise.me/biosfiles/PSX.zip</a> Try with those ;). Rename scph7001.bin bios file to scph5501.bin just as I did. Now, copy scph5500.bin, scph5502 and the new scph5501.bin that you created renaming scph7001.bin in libary/application/suport/openemu/bios. Try and tell me how you go. Good luck. Once you get the three BIOS files that OpenEmu is asking for, check to make sure the MD5 of each file matches what OpenEmu wants. [Link to OpenEmu's Github BIOS list](https://github.com/OpenEmu/OpenEmu/wiki/User-guide:-BIOS-files) Playstation (JP) Mednafen scph5500.bin 8dd7d5296a650fac7319bce665a6a53c Playstation (NA) Mednafen scph5501.bin 490f666e1afb15b7362b406ed1cea246 Playstation (EU) Mednafen scph5502.bin 32736f17079d0b2b7024407c39bd3050 This worked for me.
