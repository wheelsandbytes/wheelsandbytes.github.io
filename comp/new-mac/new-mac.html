# pro tips on setting up a new Mac (feat. jay wang) #### for those times when u get a new MacBook Pro and hate Migration Assistant... ![assistant](img/assistant.png) --- ### first things first, get [iTerm2](https://www.iterm2.com/) <!-- ![iterm 2](https://www.iterm2.com/img/logo2x.jpg) --> --- __in your preferred terminal of choice (obviously iTerm2):__ --- 1 > enable repeating keys in macOS: - `defaults write -g ApplePressAndHoldEnabled -bool false` --- 2 > install command line tools: - `xcode-select --install` --- 3 > install [Oh-My-Zsh](http://ohmyz.sh) with: - `sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"` --- ### settings to adjust in Finder: --- - go to menu bar > Finder > Preferences - match these settings: ![finder1](img/finder1.png) ![finder2](img/finder2.png) ![finder3](img/finder3.png) - then go to menu bar > View and select: - Show Status Bar - Show Path Bar ![finder4](img/finder4.png) --- ### settings to adjust in System Preferences: --- - go to System Preferences > General - change the sidebar icon size to small: ![sysprefs1](img/sysprefs1.png) - go to System Preferences > Keyboard > Modifier Keys... - change the Caps Lock key to Escape (dafuq u need caps lock for?): ![sysprefs2](img/sysprefs2.png) - if you're using multiple languages when typing, change the keyboard shortcut for input source switching to option-space: - go to System Preferences > Keyboard > Shortcuts > Input Sources - match these settings: ![sysprefs3](img/sysprefs3.png) --- ### install [BetterTouchTool](https://www.boastr.net) if u have the touch bar --- Copyright &copy; wheelsandbytes 2015-2017.
