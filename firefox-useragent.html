# How to change the user agent in Firefox ## Make the network think your Mac isn't a Mac ### 1. [Get Firefox.](https://www.mozilla.org/en-US/firefox/new/?utm_medium=referral&utm_source=firefox-com) ### 2. Open Firefox: ![Alt text](img/firefox-useragent/1.png) ### 3. In the address bar, type in `about:config` ![Alt text](img/firefox-useragent/2.png) ### 4. In the search box, type `useragent` ![Alt text](img/firefox-useragent/3.png) ### 5. Right click in the white space, and select "New" -> "String" ![Alt text](img/firefox-useragent/4.png) ### 6. Paste this into the box: `general.useragent.override` and click 'OK' ![Alt text](img/firefox-useragent/5.png) ### 7. Paste this into the box: `Mozilla/5.0 (X11; Gentoo; Linux x86_64; rv:32.0) Gecko/20150207 Firefox/32.0` and click 'OK' ![Alt text](img/firefox-useragent/6.png) ### It should look something like this: ![Alt text](img/firefox-useragent/7.png) ### 8. Turn off Firefox auto-update * When Firefox updates itself, it removes the `general.useragent.override` string, which will change the useragent back to the default agent for a Mac. We don't want that. * Go to 'Firefox' -> 'Preferences' in the menu bar * Click 'Advanced', then 'Update', and select the option that says "Never check for updates" * Disregard the security risk message, we won't be using Firefox as the main browser anyway. ![Alt text](img/firefox-useragent/8.png) ### 9. Set the homepage to Google * Click 'General' in the Preferences window * For 'Home Page:' type in `www.google.com` * This is so that Firefox will automatically try to load a webpage when you open it. Makes it a bit more convenient. ![Alt text](img/firefox-useragent/9.png) ### We're done. Anytime you see that stupid "SafeConnect" message, just open Firefox and refresh it a few times. Once Firefox can load the Google search page, you can close it and open your usual web browser. _Avast and Sophos can now go screw themselves._
